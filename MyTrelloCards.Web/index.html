Trello is a great tool that our team have been using since mid 2013. It's simple, extremely flexible and makes it easy to quickly organize many types of projects.

The only thing that has been a headache is to get a good overview everything you have been assigned to if you have multiple active boards, which is why I built MyTrelloCards using Trello's developer api. It's completely built in html/css/js.

Hope you like it!
